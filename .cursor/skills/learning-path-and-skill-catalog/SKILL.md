# Learning Path & Skill Catalog

## いつ使うか
- 新規分野の学習順序を整理したい時
- 既存スキル/テンプレの再利用を検討する時
- チーム内の学習テーマを合意形成する時

## 要点（統合）
- 学習ロードマップから「今やるべきこと」を特定する
- 技術スタック選定の理由を明文化する
- 既存スキル資産を棚卸しして再利用する

## 手順
1. 目的/プロダクト要件を確認
2. 必要な技術領域をロードマップで洗い出す
3. 既存スキル（内部/外部）を探索
4. 学習/実装の優先順位を決定

## 出力テンプレ
1. 目的 / 対象領域
2. 学習ロードマップ（短期/中期）
3. 既存スキルの候補
4. 採用理由と次アクション
