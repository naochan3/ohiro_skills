# UI/UX Resource Catalog

## いつ使うか
- 「素材」「アイコン」「フォント」「配色」「テンプレ」「UIキット」などの要望がある場合
- 参考サイトやインスピレーションの候補が必要な場合

## 参照ソース
- UI/UXリソース集（カテゴリ別のリンク集）

## 重要な制約
- プロジェクトルールが最優先
- 絵文字アイコンは禁止。SVGアイコン（Lucide/Heroicons）を使う
- 外部ストック画像はデフォルトで使わない。必要なら事前に確認する
- 画像が無い場合は生成フローを優先する
- ライセンス・商用利用可否を必ず確認する

## 作業フロー
1. 必要な素材カテゴリと用途を整理する
2. 既存アセットがあるかを確認する
3. 未提供の場合は生成 or 外部素材のどちらが適切か判断する
4. 外部素材を使う場合は候補を3〜5件に絞って提示する
5. 選定理由（用途・品質・ライセンス）を簡潔に添える

## 出力テンプレ
1. 目的 / 用途
2. 必要カテゴリ（例: アイコン, フォント, 画像）
3. 推奨候補（3〜5件）
4. 選定理由
5. ライセンス注意点
